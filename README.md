# DevTasker 

A full-stack task manager web app built as a 7-day DevOps learning project.

## Live Demo
[https://devtasker-production.up.railway.app](https://devtasker-production.up.railway.app)

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Containers:** Docker + Docker Compose
- **CI/CD:** GitHub Actions
- **Deployment:** Railway

## Features
- Add, complete, and delete tasks
- Filter tasks by All / Active / Completed
- Task creation date display
- Loading spinner
- Fully containerized with Docker
- Automated CI/CD pipeline

## Architecture
This project follows a 3-Tier Architecture:
- **Presentation Layer:** React frontend
- **Logic Layer:** Node.js + Express REST API
- **Data Layer:** PostgreSQL database
